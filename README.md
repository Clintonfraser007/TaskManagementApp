# Task Management App

## Description

A full-stack task management application using the MERN stack.

## Setup

### Backend

1. Navigate to the `backend` directory:

   ```bash
   cd backend
2. Install dependencies:
    npm install
3. Start the server:
   npm start
Frontend
1. Navigate to the frontend directory:
   cd frontend
2. Install dependencies:
   npm install
3. Start the React application:
   npm start
